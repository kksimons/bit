Security

Off by default — Bit can only run workflows you've enabled plus safe actions (open apps/URLs, Focus, move). It cannot run arbitrary shell or AppleScript, and workflows it drafts by voice stay disabled until you turn them on here. Enable Developer mode only if you want the agent to run raw commands directly.

Do Not Disturb

One-time setup (Apple requires this — Bit can't do it silently):

  1. In the Shortcuts app, click + to make a new shortcut.
  2. Add the Set Focus action → set it to Do Not Disturb, Turn On.
  3. Name it exactly Bit DND On.
  4. Make a second one: Set Focus → Do Not Disturb → Turn Off, named Bit DND Off.

Connections

Let Bit reach into other apps to answer questions like “do I have unread mail?” or “do I have open pull requests?”. Sign in once and Bit gets that app’s tools automatically.

Advanced — add a custom MCP server

For any stdio MCP server you run yourself. You’ll need its command, args, and any environment variables.

Add connection

Get your Gmail App Password (about a minute, no OAuth or setup app):

  1. Open your Google Account → Security.
  2. Turn on 2-Step Verification if it isn’t already (required).
  3. Open App Passwords (under Security).
  4. Choose Mail + any device, then Create.
  5. Copy the 16-character password and paste it above.

The password is stored locally on your Mac (never sent anywhere but Google). It can be revoked anytime from the same Google Account page.
